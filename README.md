# mcjty-website

mcjty-website

## setup

install dependencies

```shell
pnpm install
```

## formatting

format all files

```shell
pnpm format
```

## running commands

All commands for the sub projects are available in the root package.json. To run a command in a sub project, use the following syntax:

```shell
pnpm www <command>
# or
pnpm redirector <command>
```
